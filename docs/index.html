Getting Started

What are Skills?

Skills are markdown instruction files that give AI coding assistants domain-specific knowledge. They include architecture patterns, code examples, gotchas, and checklists for specific technologies.

Each SKILL.md includes:

  • YAML frontmatter — name, description, tags, compatible tools
  • Markdown body — architecture, code patterns, gotchas, checklists

Install a Skill

Use load-skill to install any skill:

npx load-skill install telegram-bot-integration

Adding a Skill

  1. Create skills/<skill-name>/SKILL.md
  2. Add YAML frontmatter with name, description, tags, compatible
  3. Write the skill content
  4. Add entry to the table in README
  5. Submit a PR
